## Usefull Link App

Heptih Frontend
<br/>

## Stack

- Typescript
- Next.js 13
- Tailwind CSS
- Deployed on Vercel
  <br/>

## Installation

```
git clone https://github.com/Asam237/usefull-link

cd frontend

yarn install

yarn dev
```
